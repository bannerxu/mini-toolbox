## 🚀 项目概述

Mini Toolbox 是一个开源的迷你工具箱项目，旨在为用户提供轻量级、便捷的工具集合。

### 核心特性

- 🎨 **多主题支持** - 6种主题可选，支持系统跟随
- 📱 **响应式设计** - 完美适配桌面端、平板和移动端
- 🔧 **模块化架构** - 组件化设计，易于扩展
- ⚡ **快速启动** - 基于 Vite 的现代化开发体验

### 技术栈

- **前端**: Vue 3 + Vite + Vue Router
- **后端**: Go
- **样式**: CSS3 + CSS 变量
- **存储**: LocalStorage
